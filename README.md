## How to Use

### 1. Prerequisites for development

- [Git LFS](https://git-lfs.github.com)
- [Node >10](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)

### 2. Setup

```bash
# Clone Repository
git clone git@github.com:cybertec-postgresql/today-i-learned.git

# Install Dependencies
yarn

# Start Development!
yarn develop
```
